REV Hub Interface

REV Hub Interface: manual rev Expansion Hub control with no progrmaing required

A port of the REV Hub Interface for all platforms, with many extra features like improved update rates and motor PIDs, and a modern user interface (in dark mode!) It runs python-tkinter as its GUI library and internally is a pyserial connection to the REV Expansion hub's FT232 chip. We hope to reimplement firmware updates at some point in the near future, however we do not currently have an ETA.

REV Hub Interface's code is public and available here.
